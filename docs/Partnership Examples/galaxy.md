# Galaxy ID

Project Galaxy is the largest Web3 credential data network in the world. Built on open and collaborative infrastructure, Project Galaxy helps Web3 developers and projects leverage credential data to build better products and communities. At the same time, data curators are rewarded when credentials are used in Galaxyâ€™s Application Modules, Credential Oracle Engine, and Credential API. 

Galaxy ID is aiming to become the Web3 LinkedIn. It implements a DID system and opens up a lot of possibilities in the world of Web3.
